AI / ML Engineer

Hi, I’m Shriyansh, an AI and Machine Learning developer focused on building data-driven and AI-powered applications. I also have experience building dashboards and forecasting models using Python, SQL, and Power BI.

I am particularly interested in LLM applications, agent workflows, vector databases, and real-world AI system design.
